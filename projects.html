Project Overview

Below are the projects I am currently working on. Click on the project names to explore more details:

  • UrbanNav - A cutting-edge autonomous navigation system using a Unitree Go1 robot dog.
  • SwimmerBot - A robotic swimmer tracker designed for athletic training and video documentation.
  • MappingNYC - An advanced urban mapping project utilizing LiDAR and GPS systems.
  • Curb2Door - A project focusing on reconstructing environments for efficient localization using LiDAR, camera, and GPS data.
  • Diffusion Models - Research on utilizing diffusion models for motion prediction and forecasting in robotic systems.
  • Biped Robot - Development of a biped robot focusing on dynamic stability and walking algorithms.
  • Visual Odometry using RGB-D Camera - Real-time camera motion estimation algorithm using RGB-D data and feature detection.
  • Variational Autoencoder for Digit Generation - Implemented a VAE to generate digit images from MNIST data.
  • Comparative Study on Optimization Techniques - Analysis of optimization methods with Adam and SGD.
  • YOLO-based Drone Detection System - Drone detection using YOLO for real-time surveillance.
  • Statistical Analysis of Glove Size Distribution - Conducted statistical studies using Python's libraries.
  • SLAM for Maze Navigation - Project using Pose Graph Mapping and feature extraction for autonomous navigation in mazes.
  • Sensor Fusion and State Estimation for Aerial Robot - State estimation using EKF and UKF for aerial robots.
  • Animetronic Hand - Gesture-controlled animetronic hand using computer vision.
  • Autonomous Robotic Systems - Multiple sub-projects including AutoInventoryBot and AutoParkBot focused on autonomous navigation and inventory management.
  • Trajectory Planning and Inverse Dynamic Control of Manipulator - Built and simulated trajectory control for UR5 manipulator.
  • Bi-Rotor Trajectory Tracker - Python-based controller for bi-rotors with trajectory tracking.
  • Robotic Arm - Basic robotic manipulator for pick and place operations.
  • Electric Car - Built a 2-seater electric car with hub motors and BLDC controllers.
