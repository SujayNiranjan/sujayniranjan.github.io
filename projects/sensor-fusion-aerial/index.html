Sensor Fusion and State Estimation for Aerial Robot

State estimation using EKF and UKF for aerial robots.

Key Features

  • Utilized EKF and UKF algorithms for state estimation in aerial vehicles.
  • Integrated data from multiple sensors for accurate localization and orientation.
  • Designed for improving autonomous navigation in GPS-denied environments.

Technologies Used

  • ROS (Robot Operating System)
  • EKF (Extended Kalman Filter)
  • UKF (Unscented Kalman Filter)
  • Sensor Fusion
