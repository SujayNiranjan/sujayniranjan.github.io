SwimmerBot: Autonomous Swimmer Tracking and Filming Robot

SwimmerBot is an autonomous robotic system designed to track and film swimmers in real-time. Using advanced computer vision techniques, it ensures high-precision tracking and smooth video capture in aquatic environments.

Key Features

  • Real-time swimmer detection and tracking.
  • Stabilized filming in aquatic conditions.
  • Compact and waterproof robotic design.

Technologies Used

  • Computer Vision
  • ROS
  • Mechanical Engineering for waterproofing
