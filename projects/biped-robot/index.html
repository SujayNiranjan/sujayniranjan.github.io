Biped Robot Development

Our project focuses on the dynamic design and gait control of a biped robot, integrating advanced robotics algorithms to enhance stability and efficiency.

Key Features

  • Utilizes feedback control mechanisms for gait stability.
  • Incorporates machine learning for gait pattern optimization.
  • Designed for adaptive walking on uneven terrain.

Technologies Used

  • ROS (Robot Operating System)
  • PID Controllers
  • Deep Learning Models
