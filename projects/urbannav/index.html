UrbanNav: Autonomous Navigation Using a Unitree Go1 Robot Dog

UrbanNav is a research project aimed at improving autonomous navigation using a Unitree Go1 robot dog. The robot is equipped with various sensors like LiDAR, cameras, and GPS to perform precise navigation tasks in urban environments.

Key Features

  • Autonomous navigation in urban settings.
  • Integration of LiDAR, cameras, and GPS sensors.
  • Real-time mapping and path planning.

Technologies Used

  • ROS
  • SLAM Algorithms
  • Computer Vision
