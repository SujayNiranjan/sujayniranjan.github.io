UrbanNav

A cutting-edge autonomous navigation system using a Unitree Go1 robot dog.

Key Features

  • Implemented autonomous navigation using LiDAR, cameras, and GPS data.
  • Developed SLAM (Simultaneous Localization and Mapping) algorithms for urban environments.
  • Integrated path planning and obstacle avoidance techniques.

Technologies Used

  • ROS (Robot Operating System)
  • SLAM (Simultaneous Localization and Mapping)
  • LiDAR, Cameras, and GPS Systems
