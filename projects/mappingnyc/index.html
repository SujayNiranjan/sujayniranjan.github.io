MappingNYC: Multimodal Urban Mapping Project

MappingNYC is a multimodal urban mapping project leveraging advanced SLAM techniques to create detailed 3D maps of urban environments. The goal is to improve autonomous navigation and urban planning.

Key Features

  • Utilizes LiDAR, GPS, and visual SLAM for precise mapping.
  • Integrates multiple sensor modalities for robust data acquisition.
  • Targets large-scale urban environments like New York City.

Technologies Used

  • ROS
  • 3D Gaussian Splatting
  • Colmap
