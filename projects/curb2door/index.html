Curb2Door: Precise 3D Reconstruction for Urban Delivery Systems

Curb2Door focuses on creating precise 3D reconstructions of urban environments to facilitate efficient delivery systems. This project explores novel techniques for mapping and localization in constrained spaces.

Key Features

  • High-resolution 3D reconstruction for urban logistics.
  • Optimized for tight and complex urban spaces.
  • Supports autonomous vehicle integration.

Technologies Used

  • SLAM
  • 3D Mapping Algorithms
  • Computer Vision
